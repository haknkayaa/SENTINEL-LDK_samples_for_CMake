Sentinel® LDK Admin API - C Sample for Linux: Readme

Overview

This sample demonstrates the use of Sentinel LDK Administration API when working in the C language.

Additional Information

Prerequisites

GNU Compiler (gcc).

Supported Operating Systems and Tested Compilers

Refer to the Sentinel LDK Release Notes - "Supported Platforms for Code Samples" and "Tested Compilers for Code Samples" sections.

Usage Notes

This section describes how to compile the sample.

Compiling the Sample

  1. Select the folder with the desired architecture.

  2. Move to the static directory. Call ./mk in the command line to build the static library samples sntl_adminapi_demo.

  3. Move to the shared directory. Call ./mk in the command line to build the shared library (.so) samples sntl_adminapi_demo.

Description of the Sample Flow

The sample consists of several functions. The default flow is as follows:

  1. Create a context to a Sentinel License Manager.

  2. Retrieve the administration information or monitoring information from the License Manager specified by the License Manager context.

  3. Set configuration options or apply actions to the License Manager specified by the License Manager context.
