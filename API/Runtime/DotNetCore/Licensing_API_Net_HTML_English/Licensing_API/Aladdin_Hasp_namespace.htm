Aladdin.Hasp Namespace

The Aladdin.Hasp namespace contains classes and types to access functionalities provided by Sentinel protection keys.

Class

Name Description

Hasp

Provides methods to access various functions of the Sentinel protection key.

HaspFile

Provides methods to access functions to read and write the memory of a Sentinel protection key.
HaspFeature Provides methods to access various attributes of Features.

HaspVersion

Provides methods to access fields of the Sentinel Licensing API Library version.

Enumeration

Name Description

HaspFileId

File IDs for default read-only and read/write memory in a Sentinel protection key.

HaspStatus

Status codes

FeatureType

Type of Feature
